## Project setup

```bash
$ npm install
```
 - I have inclued the .env content on the submission email.

## Compile and run the project
 - The app will run on [http://localhost:8000]

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm test

```
